# 手机部署指南 — Render.com 免费云部署

部署后手机会获得一个 **https://xxx.onrender.com** 地址，任何网络都能访问。

---

## 第一步：准备 GitHub（一次性）

1. 打开 https://github.com 注册账号
2. 新建仓库（New Repository），名称填 `gold-dashboard`
3. 上传整个 `价格监控` 目录下的文件

---

## 第二步：部署到 Render（一次性）

1. 打开 https://render.com 用 GitHub 账号登录
2. 点右上角 **New +** → **Web Service**
3. 选择 `gold-dashboard` 仓库
4. 填写配置：

| 配置项 | 值 |
|--------|-----|
| **Name** | gold-dashboard |
| **Runtime** | Python 3 |
| **Build Command** | `pip install -r requirements.txt` |
| **Start Command** | `python gold_server.py` |
| **Free Plan** | 勾选 Free |

5. 点 **Deploy Web Service**，等待 3-5 分钟

---

## 第三步：使用

部署完成后 Render 会给你一个地址，例如：
```
https://gold-dashboard.onrender.com
```

手机浏览器直接打开这个地址，还能**添加到主屏幕**（因为 HTTPS 支持 PWA）。

---

## 注意事项

- 免费版 15 分钟无访问会自动休眠，下次访问需等 30 秒唤醒
- 免费版每月 750 小时，足够个人使用
- 日线数据会自动保存在云端
